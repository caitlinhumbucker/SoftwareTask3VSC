Project Title: Development Project Task 3
Project Description: This website will allow users to log and view ingredients within their fridge, this program includes features such as expiry tracking, meal/ingredient planning and a nutrional value calculator.


Contact Information: caitlin.huynh1@education.nsw.gov.au
Installation Instructions: Download this repository from Github, run application, visit port 5000

Instructions for use: 
1. Register an account using a username and password, then log in. 
2. On the home page, use the "Add Ingredient Entry" form to add a new ingredient. Enter the title, description, quantity, expiry date, and nutritional values, then upload a photo. Once submitted, the ingredient will appear with all the details including a nutrional score out of 5.
3. Each ingredient card has edit and delete options. Editing lets you update details that are featured on the . Deleting removes the ingredient after a confirmation prompt.
4. Click on an ingredient to view its full details, including the nutritional breakdown and score.
